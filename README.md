# Vehicle Service Management System - Frontend

## Overview

This is the React frontend for the Vehicle Service Management System.

## Technologies Used

* React.js
* Vite
* Axios
* Recharts

## Features

* Component Management
* Vehicle Management
* Service Management
* Revenue Dashboard
* Revenue Graph Visualization

## Installation

```bash
npm install
npm run dev
```

## Backend API

Make sure the Django backend server is running before starting the frontend application.
