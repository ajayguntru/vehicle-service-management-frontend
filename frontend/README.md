# Vehicle Service Management System

## Overview

This project is a Full Stack Vehicle Service Management System developed using Django and React.

The application allows users to manage vehicle components, vehicle records, services, payments, and revenue tracking.

## Features

### Component Management

* Add Components
* View Components
* Manage Purchase Price
* Manage Repair Price

### Vehicle Management

* Add Vehicles
* View Vehicle Details
* Store Owner Information

### Service Management

* Create Service Records
* Simulate Payment Status
* Store Service Amount

### Revenue Dashboard

* Calculate Total Revenue
* Display Revenue Information

## Technology Stack

### Backend

* Django
* Django REST Framework
* SQLite

### Frontend

* React.js
* Axios
* Vite

## Installation

### Backend

```bash
cd backend
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## API Endpoints

* /api/components/
* /api/vehicles/
* /api/services/
* /api/revenue/
